INNER JOIN

INNER JOIN is a useful clause to return a result-set with records that have matching values in both tables.

For two tables, there have to have a common column share same value in order to create a linked relationship between two tables. Based on this common column, INNER JOIN can grab and only grab records that have same value to result-set.

EXAMPLE:

Table Name: one

ID Name
1 a
2 b
3 c
4 d
5 e
6 f

Table Name: two

ID Country
4 USA
5 China
6 Mexico
9 Japan
44 Australia


SQL:

SELECT one.Name, two.Country
FROM one
INNER JOIN two ON one.ID=two.ID



Result-Set

Name Country
d USA
e China
f Mexico

Note

  • The common colomn here is ID
  • In the example, INNER JOIN only grabed records that ID is 4,5,6
  • There is no a,b,c in the result set because ID 1,2,3 are not in table ‘two’
