Three Techniques

Moving Average

Using moving average the software with divide timeseries into small chunks of months each and calculate the average of each chunk, rather than the average of the entire period. At the end without moving average, the line will be pretty rough. With moving average, the line will be smoothed, which removes other relevant features (seasonality and noise) and just show the trend.

Sesonality Chart

  • Build a seasonality chart can help to show consistent and periodic fluctuations if the variable has. Seasonality is sometimes hard to detect based on raw data, but with seasonality chart, the software will analysis and compares data between each month or quarter in each year. Therefore the periodic fluctuations can be shown.
  • There is a seasonal subseries chart as well, which is often used in climate science. It gets data from different years and combined and put them into each month. So it can show how data change in years of specific month.

Index Origin

Index origin is used in zero-based index chart. The y-axis of this kind of chart shows percentage, and 0 is in the middle which is the index origin chosen by you. With different index origin the variable will show different pattern. In the book, the author first uses the average of the variable crossing multiple years as the index origin. And then uses a specific year data as the index origin. Because the difference of zero-base, the line also shows differently.
