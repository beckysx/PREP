Seeking Interaction

Interaction description

  • This visualization is a bubble chart collecting data of "World's Biggest Data Breaches & Hacks" from 2008 to 2019
  • Bubbles are floating on the screen according to different year and can move in a certain range
  • Bubbles have different radius according to how many people the affair affected
  • Bubbles can be coloredby specific filters.
  • When mouse hover on specific bubble, the bubble will expand and you can see a breif introduction. If you want to see more, click on it and it will lead you to a newspaper website.

Implement it in D3

  • I definitely will use Force Layout without edges
  • Put each affair in dataset array as an object, set attribute including Year,Impact,dataSensitivity,Genre
  • Events with same year will always floating together in a certain range using the .force("link") and force.on("tick")
  • Set .force("change") in forceSimulation function to make sure bubbles won't overlap
  • Give each botton on the screen event "on('click')" and listener respectively. When click on specific filter, the fill attribute for different circle will change
  • Give each 'g' event "on('mouseover')" and listener. When the mouse hover on one bubble, the radius will change, and text will show up
